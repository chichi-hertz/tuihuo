# 顺丰/电商退货地址截图生成器

这是一款轻量级、纯前端的单页面应用（SPA），专用于模拟生成顺丰及各大电商平台的退货寄件地址长截图。工具通过纯代码重绘了移动端的 UI 界面，生成的截图字迹清晰无毛边，排版自然。

## ✨ 核心特性

- **纯前端重绘**：抛弃了传统的“底层图片+文字覆盖”方案。整个手机预览界面均使用纯 HTML + CSS 进行了结构级的精准复刻（包含阴影、圆角、字体、渐变等细节）。
- **智能地址解析**：支持“一键智能拆分”。您可以随意粘贴诸如 `收货人：孙伟丽，电话：18621307157，地址：北京市海淀区...` 这样各种混排格式的文本，系统会通过正则和关键字判定，自动将姓名、手机号和详细地址提取并填充到对应表单。
- **实时高清预览**：左侧修改表单，右侧手机框实时同步更新效果。
- **一键下载高清长图**：内置 `html2canvas` 引擎，支持直接将排版好的 DOM 节点以 3 倍高清分辨率（Scale=3）导出并下载为 `.jpg` 格式的截图。
- **单文件、零依赖**：所需要的商品缩略图已被 Base64 编码内嵌。全项目仅需一个 `index.html` 文件，极度轻便。

## 🚀 如何使用

非常简单，您可以选择以下任意一种方式：

1. **本地使用**：
   直接双击打开下载好的 `index.html` 文件（推荐使用 Chrome 或 Edge 浏览器），即可在本地电脑上离线使用。

2. **线上部署**：
   由于本工具没有任何后端依赖，您可以直接将 `index.html` 文件推送到任意静态网站托管服务，例如：
   - GitHub Pages
   - Vercel
   - Netlify
   - 您的私人 Nginx/Apache 服务器

## 🛠 技术栈

- **HTML5 / CSS3**：用于界面绘制与响应式布局。
- **Vanilla JavaScript**：纯原生 JS 处理文本智能拆分、DOM 事件绑定。
- **[html2canvas](https://html2canvas.hertzen.com/)**：用于将网页 DOM 渲染为 Canvas 并导出为图片（通过 CDN 引入）。

## 📝 智能识别示例

在“智能识别”输入框内，您可以粘贴以下任意格式的文本进行测试：
- `北京市海淀区解放街59号阳光庭12栋6单元2501室 孙伟丽 18621307157`
- `姓名：孙伟丽，手机号码：18621307157，地址：北京市海淀区解放街59号阳光庭12栋6单元2501室`
- `孙伟丽，18621307157，北京市海淀区解放街59号阳光庭...`

点击“一键识别并拆分”后，程序会自动切分并填入对应字段中。
